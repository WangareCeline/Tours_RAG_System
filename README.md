# Tourism RAG System 

A Retrieval-Augmented Generation (RAG) system for tourism information 

## Description

This project implements a RAG system designed to provide intelligent tourism information, recommendations, and assistance. The system combines retrieval-based methods with generative AI to deliver accurate and context-aware responses about travel destinations, attractions, accommodations, and local experiences.

## Features

- **Intelligent Query Processing**: Understands natural language questions about tourism
- **Context-Aware Responses**: Provides relevant information based on user queries
- **Comprehensive Knowledge Base**: Retrieves information from tourism datasets

## Technologies Used

- Python
- RAG (Retrieval-Augmented Generation) Architecture
- ChromaDB.
